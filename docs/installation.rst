============
Installation
============

Requirements:
  - python >= 2.7

You can install, upgrade, uninstall dbcut with these commands::

  $ pip install [--user] dbcut
  $ pip install [--user] --upgrade dbcut
  $ pip uninstall dbcut

Or from git (last development version)::

  $ pip install git+https://github.com/SalemHarrache/python-dbcut.git

Or if you already pulled the sources::

  $ pip install path/to/sources

Or if you don't have pip::

  $ easy_install dbcut
